== Description ==
A plugin to create a shortcode for displaying FAQs with accordion functionality. Easily manage FAQs using the custom post type and display them on any page or post using a simple shortcode.

== Installation ==
1. Upload the `shortcode-faq-manager` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Create your FAQs using the custom post type created by the plugin.
4. Use the shortcode `[faqs]` to display the FAQs on any page or post.

== Usage ==
- Use the shortcode `[faqs count="5"]` to display a specified number of FAQs.
- If the `count` attribute is not provided, all FAQs will be displayed.

== Frequently Asked Questions ==

= How do I create FAQs? =
Once the plugin is activated, you can add FAQs by navigating to the "FAQs" section in the WordPress admin menu.

= Can I customize the styles? =
Yes! You can customize the styles by editing the `faq-accordion.min.css` file located in the `assets/build/css` directory.
